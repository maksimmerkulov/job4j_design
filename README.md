# job4j_design

[![Java CI with Maven](https://github.com/maksimmerkulov/job4j_design/actions/workflows/maven.yml/badge.svg)](https://github.com/maksimmerkulov/job4j_design/actions/workflows/maven.yml)

## About the Project

This project covers the following topics:

1. Data Structures and Algorithms:
    - Iterator
    - Generic
    - List
    - Set
    - Map
    - Tree

2. Input-Output:
    - Streams
    - Socket
    - Logging
    - Serialization

3. SQL, JDBC:
    - PostgreSQL
    - Create, Update, Insert
    - Query
    - Join
    - Database Objects
    - JDBC
    - Liquibase

4. Garbage Collection:
    - Garbage Collection Concept
    - Types of Garbage Collectors
    - Application Profiling
    - Reference Types and Soft/Weak Reference Collections
    - Java Memory Model

5. Clean Architecture:
    - TDD
    - SRP
    - OCP
    - LSP
    - ISP
    - DIP

6. Interview Algorithms:
    - Two Pointers
    - Hash Structures
    - Sorting
    - Queues and Stacks
    - Sliding Window
    - Greedy Algorithm
    - Trees
